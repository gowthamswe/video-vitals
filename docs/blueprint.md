# **App Name**: TubeSleuth

## Core Features:

- Clickbait Flag: Add a 'Clickbait' button to YouTube videos.
- Information Density Slider: Add a slider to control the information density, with values from 0-10.
- Rating Storage: Save clickbait and information density votes and ratings for each video to Firestore.

## Style Guidelines:

- Primary color: Muted blue (#6699CC) to blend with YouTube's UI while still standing out.
- Background color: Light gray (#F0F0F0) for a clean, unobtrusive look.
- Accent color: Soft orange (#E59866) for interactive elements like the slider and button.
- Font: 'Inter' sans-serif, for a modern and neutral feel suitable for UI elements.
- Use simple, clear icons for the clickbait flag and information density controls.
- Place the 'Clickbait' button and information density slider directly beneath the YouTube video, next to the like/dislike buttons.
- Subtle animation when the clickbait button is clicked to provide feedback to the user.
